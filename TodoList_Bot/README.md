# TODO-List Telegram Bot with Python-telegram-bot

## Commands:
- <B>/add taskName </B> (to add a task)
- <B>/list </B> (to view tasks list)
- <B>/check i </B> (to check task(i) as done)
- <B>/clear </B> (to clear all tasks)
